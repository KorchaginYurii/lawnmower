ГАЗОНОКОСИЛКА.
К сожалению, спонсоры закрыли проект Autonomous Cabbage Robot Platform. Зато есть новый проект lawnmower.
Небольшие изменения в конструкции робота привели к новым данным по потребляемой мощности: 
MOVE_COST = 0.05
TURN_COST = 0.02
CUT_COST = 0.45
но возникают проблемы с навигацией - если с капустой все было просто: 1 клетка - 1 захват капуст, навигация была по меткам на поле, то теперь надо убирать траву так, чтобы не было пробелов по краям клетки (но это можно, наверное решить уменьшением размера клетки так, чтобы она была меньше размера робота и он гарантированно покосит границу клетки). Гораздо более существенная проблема - это не прямоугольные препятствия - круглые, овальные, криволинейные, как решать эту проблему не знаю. Какие способы навигации существуют по цена/качество? Посмотри на карту тестового участка размером 45х42 метра, заштрихованную область необходимо косить, размер робота 0.5х0.5 метра.
